# VIGIL Frontend

Frontend for VIGIL — a real-time observability platform for developers.

Provides dashboards for monitoring logs, events, uptime, and alerts.

---

## Tech Stack

- React (Vite)
- React Router v6
- Axios
- Socket.io Client
- Zustand (or Redux optional)
- Tailwind / custom styling

---

## Features

- Authentication UI
- Watch Page (API Monitoring)
- Relay Page (Webhook events)
- Stream Page (Live logs)
- API Key Management
- Alert UI
- Real-time updates via WebSockets

---

## Setup Instructions

### 1. Clone repo

```bash
git clone https://github.com/your-username/vigil-frontend.git
cd vigil-frontend
```

---

### 2. Install dependencies

```bash
npm install
```

---

### 3. Setup environment variables

Create `.env`:

```env
VITE_API_URL=http://localhost:5001/api
VITE_SOCKET_URL=http://localhost:5001
```

---

### 4. Start frontend

```bash
npm run dev
```

---

## App Routes

| Route   | Description        |
| ------- | ------------------ |
| /login  | Login page         |
| /watch  | API monitoring     |
| /relay  | Webhook relay      |
| /stream | Live logs          |
| /keys   | API key management |

---

## API Integration

Axios base URL:

```js
http://localhost:5001/api
```

---

## Real-time (Socket.io)

Connects to:

```js
http://localhost:5001
```

Events:

- `log:new`
- `event:received`

---

## Deployment

### Frontend → Vercel

1. Push to GitHub
2. Import project in Vercel
3. Set env variables:
   - `VITE_API_URL`
   - `VITE_SOCKET_URL`

4. Deploy
